# Matrix

matrix in terminal made in JavaScript

## Use 
**git clone**
```
git clone https://github.com/CamiaCL/terminalMatrix.git
```

**Enter the directory**
```
cd terminalMatrix
```

**Run**
```
node matrix.js
```
